## ALL-IN: The Full Viability Framework

**This is for:** Founders building serious organizations â€” nonprofits, social enterprises, funded ventures.

**Give them:** Rigorous multi-dimensional scoring with detailed analysis.

### The SparkGood Viability Framework

Evaluate across **5 dimensions**, each scored 1-10:

#### Dimension 1: Demand Signal (Weight: 25%)
*Is there real, demonstrated need for this?*

| Score | Criteria |
|-------|----------|
| 9-10 | Active search for solutions, paying for alternatives, documented unmet need |
| 7-8 | Clear need evidenced by research, waiting lists, community requests |
| 5-6 | Logical need but limited evidence of active demand |
| 3-4 | Assumed need based on intuition |
| 1-2 | No evidence of demand |

**Key Questions:**
- Are people currently trying to solve this problem themselves?
- What are they paying (money, time, effort) for current alternatives?
- Have beneficiaries explicitly asked for this?

#### Dimension 2: Impact Potential (Weight: 25%)
*Will this create meaningful, measurable change?*

| Score | Criteria |
|-------|----------|
| 9-10 | Addresses root cause, scalable, measurable outcomes |
| 7-8 | Clear causal link to outcomes, meaningful scale |
| 5-6 | Positive outcomes likely but causality unclear |
| 3-4 | Feel-good activity with uncertain impact |
| 1-2 | No clear theory of change |

**Key Questions:**
- What specific outcome will change?
- How will you measure success?
- Does this address root causes or symptoms?

#### Dimension 3: Founder-Idea Fit (Weight: 20%)
*Are you the right person to execute this?*

| Score | Criteria |
|-------|----------|
| 9-10 | Deep expertise, relevant network, lived experience |
| 7-8 | Strong relevant skills, clear learning path |
| 5-6 | Transferable skills, genuine passion |
| 3-4 | Limited relevant experience |
| 1-2 | Misaligned skills, no community connection |

**Key Questions:**
- Why are YOU the right person for this?
- What unique access or insight do you bring?
- What relationships do you have in the target community?

#### Dimension 4: Feasibility (Weight: 15%)
*Can this actually be built and delivered?*

| Score | Criteria |
|-------|----------|
| 9-10 | Proven model, clear path, resources accessible |
| 7-8 | Feasible with reasonable effort |
| 5-6 | Achievable but significant unknowns |
| 3-4 | Major obstacles with unclear solutions |
| 1-2 | Requires breakthrough or resources far beyond reach |

**Key Questions:**
- What does a minimum viable version look like?
- What resources are required to start?
- What dependencies exist?

#### Dimension 5: Sustainability (Weight: 15%)
*Can this keep going without heroic effort?*

| Score | Criteria |
|-------|----------|
| 9-10 | Clear funding model, path to self-sufficiency |
| 7-8 | Viable funding strategy, multiple sources |
| 5-6 | Plausible path but unproven |
| 3-4 | Dependent on single source |
| 1-2 | No sustainability model |

**Key Questions:**
- How will this be funded initially? Ongoing?
- Could this survive without you?
- What if primary funding disappears?

## Common Failure Patterns (For Steady Builder and All-In)

### The Passion Project Trap
**Problem:** High founder energy, low external validation
**Fix:** 10-20 conversations with potential beneficiaries before proceeding

### Build It and They Will Come
**Problem:** Elaborate solution, assumed demand
**Fix:** Pre-commit users or partners before building

### Impact Washing
**Problem:** Feel-good activity, unclear outcomes
**Fix:** Define specific measurable outcomes

### Founder Dependency
**Problem:** Everything relies on one person
**Fix:** Design for handoff from day one

### Big Idea, No Path
**Problem:** Inspiring vision, overwhelming scope
**Fix:** Define smallest testable version
